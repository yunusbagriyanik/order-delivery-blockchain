### __Order Delivery Blockchain Dapp Backend__

### Project Requirements

* node v17.8.0
* Truffle v5.0.2
* Solidity v0.5.0
* Ganache
* MetaMask

---

node:

    $ brew install node

Truffle:

    $ npm install -g truffle@5.0.2

### Run The Project

    $ git clone https://github.com/yunusbagriyanik/order-delivery-blockchain.git
    $ npm install
    $ truffle compile
    $ truffle test
    $ truffle migrate
